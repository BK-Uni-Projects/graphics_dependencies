Dependencies for OpenGL projects in C++

- https://www.libsdl.org/[SDL2]

- http://glew.sourceforge.net/[GLEW: The OpenGL Extension Wrangler Library]
  * GLEW 1.13.0
- http://glm.g-truc.net[OpenGL Mathematics (GLM) ]
  * GLM 0.9.7.1
